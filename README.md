# my_pantry
MyPantry web application code
## Installation Instructions
1. clone the repo <br/>
```console
git clone git@github.com:kai-OB/my_pantry.git
```
2. cd into project directory <br/>
```console
cd my_pantry
```
3. get project dependencies
```console
npm install
```
4. run the server
```console
npm start
```
