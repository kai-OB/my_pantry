# my_pantry
MyPantry web application code
## CSE 115A Instructions
1. clone the repo <br/>
```console
git clone git@github.com:kai-OB/my_pantry.git
```
2. cd into project directory <br/>
```console
cd my_pantry
```
3. install express
```console
npm install express --no-save
```
4. run the server
```console
node app.js
```
